GradMemoir

A platform for graduates to share experiences and insights, combining the best features of Instagram & LinkedIn.

Problem: New students struggle to connect with seniors, explore their experiences, and find guidance on college life, events, and career paths. Graduates, in turn, want a place to document their journey, cherish memories with friends, and revisit their college experiences while staying connected with juniors.

Approach: Developed a social networking platform using the MERN stack, enabling graduates to share insights and connect with juniors.

Tech Stack: React.js, Node.js, Express, MongoDB, JWT Authentication.

Challenges: Implementing secure authentication, optimizing database queries, and ensuring a scalable backend.

Outcome: Successfully onboarded over 100 users in the first beta release, fostering meaningful connections between students and graduates.

Multi-Threaded Proxy Web Server

A high-performance proxy web server built in C, implementing LRU caching for optimized response time.

Problem: Needed an efficient proxy server to manage multiple client requests.

Approach: Developed a multi-threaded server with LRU caching for faster responses.

Tech Stack: C, POSIX threads, LRU Caching.

Challenges: Implementing thread synchronization to avoid race conditions.

Outcome: Improved request handling speed by 50% compared to a single-threaded model.

DevTinder

A Tinder-like platform for developers to match based on skills and interests, fostering collaboration on projects.

Problem: Developers struggle to find like-minded collaborators for projects.

Approach: Created a match-based system using Node.js and MongoDB.

Tech Stack: Node.js, Express, MongoDB, Socket.io, WebSockets.

Challenges: Implementing real-time matching and message synchronization.

Outcome: Users were able to find collaborators efficiently with an intuitive matching system.
